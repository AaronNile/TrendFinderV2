# TrendFinderV2

TrendFinder is a Python-based web application that analyzes stock price trends using moving averages and volatility-based bounds to identify potential trend signals.

## Live Demo

Deployed on Render:
https://trendfinderv2.onrender.com

## Overview

TrendFinder allows users to:

- Enter a stock ticker symbol
- Retrieve historical market data (via Yahoo Finance)
- Compute moving averages
- Generate upper and lower volatility bounds
- Visualize trend behavior over time

The application is designed to explore how stock volatility and moving averages can be used to forecast the direction of price-action.
